# SUPERMIND — конвейер стратегического брейншторма

Скилл для Claude, собранный из 27 сильнейших систем мышления мира (Cynefin,
12 точек рычага Медоуз, ТРИЗ, Теория ограничений Голдратта, карты Уордли,
суперпрогнозирование Тетлока, СМД Щедровицкого и др.) в один конвейер:

**гейт-анкета → позиция → диагноз → структура → разрыв рамки → атака → зонд**

Формула: гениальность = точность прицела × дерзость гипотез × жестокость
отбора. Конвейер не производит гениальность по кнопке — он делает банальность
маловероятной, а сильную идею статистически достижимой.

## Установка

### Claude Code — автоустановка по ссылке (рекомендуется)

```
/plugin marketplace add ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ
/plugin install supermind@supermind-skills
```

Обновление до новых версий: `/plugin marketplace update supermind-skills`.

### Claude.ai (веб и мобильный)

Скачай `dist/supermind.skill`, отправь файл в чат с Claude и нажми
**Save skill** на карточке файла. Либо загрузи через Настройки → Возможности
(Skills). Актуальные пути интерфейса: https://support.claude.com

### Вручную (Claude Code, без маркетплейса)

```
git clone https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ
cd ИМЯ_РЕПОЗИТОРИЯ && ./install.sh
```

## Как пользоваться

Скажи Claude: «Прогони через Supermind: …» — или просто принеси сложную
задачу. Подробное руководство со всеми режимами — [ИНСТРУКЦИЯ.md](ИНСТРУКЦИЯ.md).

## Структура

```
.claude-plugin/marketplace.json     — каталог для автоустановки
plugins/supermind/                  — плагин
  .claude-plugin/plugin.json        — манифест
  skills/supermind/                 — сам скилл
    SKILL.md                        — ядро: конвейер, фазы, защиты
    references/                     — 27 досье + 6 протоколов
    evals/evals.json                — 9 регрессионных тестов
dist/supermind.skill                — пакет для Claude.ai в один клик
install.sh                          — ручная установка
```
