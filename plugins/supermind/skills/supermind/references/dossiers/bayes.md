# Байесовское обновление (Томас Байес, Лаплас; современная рационалистская традиция)

**Суть.** Убеждение — не да/нет, а распределение вероятностей, обязанное меняться по правилу при каждом свидетельстве: P(H|E) ∝ P(E|H)·P(H). Сила свидетельства — отношение правдоподобий: насколько E вероятнее в мире, где H истинна, чем в мире, где ложна.

**Когнитивная операция.** Квантованная ревизия: каждое наблюдение обязано сдвинуть уверенность на вычислимую величину — или признаётся неинформативным.

**Механика.** 1) Прайор до данных (честный, лучше из базовой частоты). 2) Для нового факта спроси: насколько он ожидаем, если я прав? если не прав? 3) Ключевой тест диагностичности: «этот факт удивил бы меня в мире, где я ошибаюсь?» Если нет — факт не свидетельство (восторг ранних пользователей ожидаем в обоих мирах — он почти ничего не весит). 4) Сдвинь уверенность пропорционально. 5) Заранее назови, какое наблюдение сдвинуло бы тебя вниз, — иначе гипотеза нефальсифицируема для тебя.

**Вход → выход.** Поток фактов → взвешенные сдвиги уверенности вместо «подтверждается!» на каждый приятный сигнал.

**Сильна.** Интерпретация результатов зондов/тестов; вакцина от подтверждающего искажения.

**Ломается.** Числовой фетишизм без данных; прайоры из воздуха.

**Уникальный вклад.** Тест диагностичности свидетельства — «в каком мире этот факт вероятнее?» — одна строка, убивающая 80% самообмана при чтении «сигналов рынка».
