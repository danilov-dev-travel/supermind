# Синектика (Уильям Гордон, Джордж Принс, 1950-е)

**Суть.** Творчество = «сделать незнакомое знакомым и знакомое — незнакомым». Двигатель — дисциплинированная аналогия четырёх типов: прямая (как эту задачу решает природа/другая отрасль?), личная (я — этот объект: что я чувствую?), символическая (парадоксальное сжатие: «мягкая твёрдость»), фантастическая (как это решилось бы в мире магии?).

**Когнитивная операция.** Перенос структуры решения из далёкого домена через мост аналогии.

**Механика.** 1) Сформулируй функцию задачи абстрактно («соединять не повреждая», «удерживать внимание без принуждения»). 2) Прямые аналогии из ≥3 удалённых доменов: биология, военное дело, музыка, спорт, религия. 3) Личная: побудь самим продуктом/клиентом/багом — что тебе мешает? 4) Символическая: сожми суть в оксюморон, раскрути его. 5) Верни находки в задачу: что из механики донора переносится?

**Вход → выход.** Функциональная формулировка → 3–5 механизмов-доноров из чужих доменов → адаптированные концепции.

**Сильна.** Классика переносов: липучка ← репейник, вокзальная логистика ← муравьи; там, где своя отрасль исчерпана.

**Ломается.** Поверхностные аналогии (по виду, не по механизму); требует широкой эрудиции — идеально для LLM.

**Уникальный вклад.** Личная аналогия — эмпатическая симуляция изнутри объекта: странный, но регулярно пробивающий приём, которого нет больше нигде.
