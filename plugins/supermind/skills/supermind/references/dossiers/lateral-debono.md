# Латеральное мышление (Эдвард де Боно, 1967+)

**Суть.** Мозг — самоорганизующаяся паттерн-система: мысль скатывается в проторённые желоба. Вертикальное мышление углубляет ту же яму; латеральное — копает в другом месте. Инструменты: провокация (PO), случайный вход, движение (movement) вместо суждения, шесть шляп.

**Когнитивная операция.** Управляемый разрыв паттерна: сознательно занять нелогичную позицию, чтобы с неё увидеть новый маршрут.

**Механика.** 1) Провокация: сформулируй заведомый абсурд («PO: у ресторана нет еды»). 2) Движение: не оценивай, а извлекай — к чему это ведёт? какой принцип внутри? («клиенты приносят своё → мы продаём место и атмосферу → коворкинг-ужины»). 3) Случайный вход: возьми случайное слово и принудительно построй мост к задаче. 4) Шесть шляп: разнеси режимы (факты/эмоции/риски/выгоды/креатив/процесс) по времени — параллельное мышление вместо спора.

**Вход → выход.** Затоптанная задача → 3–5 идей из желобов, куда логика не заглядывает.

**Сильна.** Когда все идеи «правильные и одинаковые»; групповой формат (шляпы гасят статусные войны).

**Ломается.** Без фазы «движения» провокации остаются шутками; на точно поставленных противоречиях ТРИЗ бьёт прицельнее.

**Уникальный вклад.** Легализация нелогичного шага как *техники*: PO — официальная лицензия говорить абсурд с обязательством извлечь из него принцип.
