# Морфологический анализ (Фриц Цвикки, 1940-е)

**Суть.** Решение — точка в многомерном пространстве параметров. Вместо блуждания — построить всё пространство: выделить независимые параметры задачи, для каждого выписать варианты значений, перебрать комбинации. Цвикки так «изобретал» типы реактивных двигателей — включая ещё не существовавшие.

**Когнитивная операция.** Комбинаторная развёртка: заменить «придумывание» систематическим произведением множеств.

**Механика.** 1) Выдели 3–6 существенных параметров задачи (для сервиса: кто платит × за что × канал × момент × модель цены). 2) Для каждого — 3–6 значений, включая экзотические. 3) Построй ящик; всего комбинаций = произведение. 4) Вычеркни внутренне противоречивые. 5) Пройди неочевидные клетки: каждая — кандидат в концепцию. Особое внимание пустым клеткам рынка: комбинация возможна, но никем не занята.

**Вход → выход.** Задача → параметрическая карта всего пространства решений + список незанятых клеток.

**Сильна.** Дизайн продуктов/бизнес-моделей/стратегий; гарантирует полноту охвата — не «что пришло в голову», а «всё, что бывает».

**Ломается.** Комбинаторный взрыв при >6 параметрах; качество упирается в выбор осей.

**Уникальный вклад.** Единственный метод с гарантией полноты: показывает решения, до которых ассоциативное мышление не дотягивается в принципе.
