# Журнал решений и возврат зонда

Без памяти каждый зонд уходит в пустоту: розовые отчёты принимаются на веру,
калибровка не растёт, скилл не умнеет. Журнал и протокол возврата закрывают
петлю.

## Журнал: формат

Одна сессия = одна запись. При первом финале предложи создать файл
«журнал-решений.md» (артефакт/документ пользователя) и дальше дописывай:

```
## [дата] — [тема в 3–5 словах]
Вопрос (переформулированный): …
Противоречие: …
Финалисты: A …, B …, C …
Зонд: … | сигнал «работает»: … | сигнал «стоп»: …
Прогноз: [полоса или % с источником]
--- после возврата ---
Результат: … | вердикт: работает / стоп / мутно
Калибровка: прогноз [X] → факт [попал / оптимист / пессимист]
Рамка: что поняли о постановке
```

Журнал — техническая память конвейера: противоречие формулой, уровень рычага
и залпы-источники пишутся сюда как есть, даже когда пользователю всё подано
простым языком (`plain-language.md`). Так строгость сохраняется, а разговор
остаётся человеческим.

В начале новой сессии: тема похожа на продолжение → спроси про журнал или,
если доступен поиск по прошлым чатам, найди прошлую запись сам. Начинать с
нуля то, у чего есть журнал, — ошибка.

## Протокол возврата зонда (мини-интейк результатов)

Пользователь принёс результат — отчёт не принимается как факт:

1. **Наблюдения vs впечатления.** «Что именно произошло — эпизоды, цифры,
   дословные фразы?» «Всем понравилось» — впечатление; «6 из 20 заплатили
   полную цену» — наблюдение.
2. **Вопрос скептика.** «Что в этом отчёте оспорил бы недоброжелатель?»
3. **Диагностичность (Байес).** Случилось бы это и в мире, где идея мертва?
   Покупки со скидкой, вежливый восторг, лайки — недиагностичны, в зачёт
   не идут.
4. **Сверка с заранее названными сигналами.** Попали в «работает»? В «стоп»?
   Ни туда ни туда — зонд был размытым: признай это прямо.
5. **Пересмотр карты.** Часть сменила тип (комплексное прояснилось →
   сложное)? Рамка держится? Три возврата подряд «мутно» — чини не зонды,
   а рамку: возврат в фазу 0 (двойная петля).
6. **Запись в журнал + калибровка.**

## Счёт калибровки

Каждые ~5 записей — короткая рефлексия по журналу: где систематический
оптимизм (прогноз «высокая» → факт «стоп»)? Какие залпы дают выживающие
идеи, а какие — красивые трупы? Выводы пиши в шапку журнала как «личные
поправочные коэффициенты» и учитывай в следующих прогнозах. Это механизм,
которым методология умнеет под конкретного пользователя.
